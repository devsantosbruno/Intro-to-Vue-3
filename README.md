## Tecnologias utilizadas


•   HTML

•   Css

•   JavaScript

•   Vue.js


## Rodar local

Clone o projeto

```bash
  git clone git@github.com:devsantosbruno/Intro-to-Vue-3.git
```

Vá para o diretório do projeto

```bash
  cd Intro-to-Vue-3
```

Instale as dependências

```bash
  npm install
```

Compile a estilização do projeto

```bash
  npm run dev
```
